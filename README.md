#### Prerequisites
JDK 8+ 

## Development
#### Getting Started
Clone this repository.

```bash
git clone --recursive https://github.com/ParNote-App/ParNote.git
cd ParNote
```

##### Compile & Run & Debug

```bash
./gradlew vertxDebug
```
